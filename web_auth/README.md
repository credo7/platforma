# pgp-web

# Install

```
# packages
sudo apt install nodejs
sudo npm install --global yarn

# setup
yarn setup

# copy env
yarn env:copy

# run production
yarn pm2:prod

# run development
yarn pm2:dev

# build production
yarn build
```

## Notice

### development

- web: http://localhost:3000

### production

- web: http://croc.hopto.org:5000

### build

- config web server to folder './build'
